# BullChat Room Visual System

## Purpose

This document defines the complete visual identity system for every room in BullChat.

BullChat rooms are **not** generic chat channels. Every room is treated as its own destination with a unique atmosphere, story, artwork, and personality while remaining part of the BullChat universe.

The objective is simple:

> Users should understand what a room represents and how it feels before reading a single message.

Every room should immediately communicate its purpose through world-class visual storytelling.

---

# Design Philosophy

BullChat is more than a messaging platform.

It is a collection of carefully designed worlds where every room has its own identity, atmosphere, and story.

When users move between rooms, they should feel like they are entering different places—not simply switching chat channels.

Examples:

- Headquarters
- Trading Floor
- Innovation Lab
- Creative Studio
- Broadcast Center
- Research Facility
- Launch Bay

---

# Core Principles

## 1. Every Room Is A Place

Rooms should feel like real destinations.

Every environment should have its own purpose.

Examples:

- Headquarters
- Trading Floor
- Creative Studio
- Startup Lab
- Broadcast Studio
- Research Center

---

## 2. Cinematic First

Artwork should feel like scenes from a AAA game or blockbuster movie.

Never create artwork that looks generic.

### Never

- Clipart
- Flat icons
- Emojis
- Cartoon stickers
- Generic AI wallpaper
- Bright colorful illustrations

### Instead

- Realistic digital artwork
- Cinematic lighting
- Strong composition
- Premium rendering
- Story-driven environments
- Dark immersive atmosphere

---

## 3. Storytelling

Every artwork should answer one question:

> "What happens inside this room?"

Users should understand this within **3 seconds**.

Every room tells its own story.

---

## 4. Consistent Universe

Although every room is unique,

every room belongs to BullChat.

Every artwork should share:

- Same premium quality
- Same rendering style
- Same dark atmosphere
- Same green accent color
- Same level of detail
- Same visual language

BullChat should feel like one connected universe.

---

# Asset Structure

Every room must contain the following visual assets.

---

## 1. Hero Cover (Desktop)

Purpose

Primary desktop room banner.

Used for:

- Room landing page
- Desktop room header
- Featured room presentation

Recommended Size

```
1920 × 640
```

Requirements

- Cinematic
- Strong focal point
- Plenty of negative space for UI
- No text baked into artwork

---

## 2. Hero Cover (Mobile)

Purpose

Primary mobile room banner.

Used for:

- Mobile room header

Recommended Size

```
1080 × 1350
```

Requirements

- NOT simply cropped
- Specifically composed for vertical devices
- Preserve important subjects

---

## 3. Room Avatar

Purpose

Primary room identity.

Used throughout BullChat.

Examples

- Sidebar
- Room List
- Search
- Notifications
- Trending Rooms
- Recently Active
- Mobile Lists

Recommended Size

```
512 × 512
```

Requirements

- Simple
- Instantly recognizable
- Strong central subject
- Readable at 32 × 32

---

## 4. Room Thumbnail

Purpose

Discovery asset.

Used for:

- Explore
- Featured Rooms
- Trending Rooms
- Suggested Communities
- Home Feed
- Search Results

Recommended Size

```
1280 × 720
```

Requirements

- Promotional
- High impact
- Encourages discovery
- Dynamic composition

Think of this as the movie poster for the room.

---

## 5. Background Pattern

Purpose

Ambient room texture.

Used behind:

- Loading Screens
- Empty States
- Room Settings
- Welcome Screens
- Skeleton Placeholders
- Modals
- Information Panels

Recommended Size

```
1920 × 1080
```

Requirements

- Very subtle
- Atmospheric
- Never competes with UI
- Should enhance—not distract

# Composition Guidelines

## Hero Covers

Should include:

- Large environment
- Strong depth
- One dominant focal point
- Space for room title overlay
- Room atmosphere

Avoid:

- Busy compositions
- Tiny details everywhere
- Text
- Logos floating randomly

---

## Mobile Covers

Must be redesigned—not cropped.

The most important subjects should remain centered.

---

## Avatars

Should focus only on the defining subject.

Examples:

- Guardian Bull
- Trading Monitor
- Podcast Microphone
- Startup Workstation

Avoid wide environments.

---

## Thumbnails

Must communicate:

> "Why should I enter this room?"

within seconds.

They are promotional assets.

---

## Background Patterns

Should feel invisible.

Users should notice the atmosphere—

not the artwork itself.

---
