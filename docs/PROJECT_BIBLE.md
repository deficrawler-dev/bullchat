# BullChat Project Bible

## Overview

BullChat is a premium Web3 community platform designed to become the home of crypto communities on X.

BullChat launches with a focused mission: building the best possible home for Ansem's community. Rather than trying to serve everyone from day one, BullChat starts with a single community and expands over time.

BullChat is built for people who want to connect, collaborate, discover opportunities, and grow within the Web3 ecosystem.

---

# Vision

Build the platform where Web3 communities naturally gather to chat, network, collaborate, find opportunities, and grow together.

---

# Mission

Help Web3 communities connect, collaborate and build.

---

# Tagline

Connect. Collaborate. Build.

---

# Launch Strategy

## Phase 1

Launch exclusively for Ansem's community.

Everything in Version 1 should improve the experience of Ansem's followers.

## Phase 2

Expand to the wider crypto community on X.

## Phase 3

Allow other creators and communities to build their own spaces inside BullChat.

---

# Core Principles

Every feature must satisfy at least one of these goals.

- Help users connect.
- Help users collaborate.
- Help users build.
- Help users discover opportunities.
- Strengthen the community.

If a feature does not satisfy one of these goals, it should not be added.

---

# Product Philosophy

BullChat is not designed to replace Discord.

BullChat is not designed to replace X.

BullChat complements both by providing a dedicated space where communities can build deeper relationships and discover opportunities beyond social feeds.

---

# Long-Term Goal

BullChat should become the default community platform for Web3 creators and communities.
