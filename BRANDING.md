# BullChat Branding Guide

Version: 1.0

---

# Brand Mission

BullChat exists to become the professional home for Web3 communities.

Our mission is to provide a modern, secure and intelligent platform where crypto communities can communicate, collaborate and grow.

Version 1 is dedicated to the $ANSEM community.

Future versions will support additional communities without compromising quality.

---

# Brand Personality

BullChat should always feel:

• Professional
• Confident
• Intelligent
• Modern
• Trustworthy
• Community Driven
• Bullish
• Premium

Never feel:

• Childish
• Loud
• Meme-heavy
• Flashy
• Over-designed
• Corporate

---

# Brand Values

Every design and engineering decision should reinforce:

Trust

Transparency

Speed

Quality

Community

Security

Scalability

---

# Brand Voice

BullChat communicates like an experienced builder.

Tone should be:

Confident

Friendly

Direct

Helpful

Clear

Professional

Never:

Overhype

Use excessive emojis

Sound like marketing copy

Use unnecessary buzzwords

# Naming

Correct

BullChat

Incorrect

bullchat

Bull Chat

Bullchat

Always capitalize both words.

---

# Logo

BullChat's logo is the strongest visual representation of the brand.

It should never be redesigned without approval.

The logo consists of:

• Bull icon
• BullChat wordmark

The icon may be used independently where appropriate.

---

# Logo Usage

Allowed

• Website header
• Mobile app
• Admin dashboard
• Marketing material
• Social media
• Documentation

Avoid

Stretching

Rotating

Changing colors

Adding shadows

Adding outlines

Changing proportions

---

# Logo Spacing

Maintain clear space around the logo.

Minimum spacing should equal approximately the height of the bull's nose.

No other elements should enter this area.

---

# Minimum Logo Size

Desktop

40px+

Mobile

32px+

Never display the logo smaller than these values.

# Color Philosophy

Green is an accent.

It should create focus, not dominate the interface.

Recommended usage:

90%

Dark neutral colors

10%

Brand Green

The interface should feel calm, while green draws attention to important actions.

Minimal

Modern

Technical

Clean

Crypto-inspired

Cartoon illustrations

Gaming artwork

Comic styles

Heavy gradients

---

# Photography

If photography is used:

Use authentic imagery.

Avoid generic corporate stock photos.

Prefer screenshots of BullChat over unrelated visuals.

Fade

Slide

Scale

Duration

150–250ms

# Marketing Style

BullChat marketing should emphasize:

Community

Speed

Quality

Security

Professionalism

Build trust before excitement.

---

# Writing Style

Prefer:

"Join the conversation."

"Build with your community."

"Discover opportunities."

Avoid:

"To the moon."

"100x."

"Guaranteed gains."

"Hype-first messaging."

BullChat is a platform, not a token promotion.

---

# Social Media

Posts should be:

Educational

Product-focused

Community-focused

Transparent

Avoid excessive engagement bait.

---

# Screenshots

Screenshots used publicly should:

Show realistic conversations.

Avoid placeholder lorem ipsum.

Never expose private information.

Use production-quality UI only.

---

# Accessibility

Branding should remain accessible.

Maintain sufficient color contrast.

Avoid relying only on color to communicate meaning.

---

# Future Branding

As BullChat expands to multiple communities:

The BullChat identity remains constant.

Community-specific branding should complement the BullChat brand—not replace it.

BullChat should always be recognized first.

---

# Brand Promise

BullChat is where serious Web3 communities connect, collaborate and grow.

Every screen, interaction and message should reinforce that promise.

---

# Claude Branding Rules

Claude must never redesign the BullChat logo.

Claude must never introduce unapproved colors.

Claude must follow DESIGN_SYSTEM.md for every UI component.

Claude must maintain consistent spacing, typography and branding across the application.

Claude must preserve BullChat's premium identity in every new feature.
